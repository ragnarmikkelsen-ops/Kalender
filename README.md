# Kalender

Mine kalendre, gjerne for deling.

## Timeplan.ics

Et lokalt script (`Timeplan_Watch_004.py`) leser timeplan-PDF-er fra
Visma og genererer `Timeplan.ics` automatisk. Filen pushes hit til
GitHub, slik at den kan brukes som et **abonnement** i kalenderapper.

### Slik abonnerer du

Bruk denne URL-en i Outlook, Google Calendar, iPhone, Android osv.:

```
https://raw.githubusercontent.com/ragnarmikkelsen-ops/Kalender/main/Timeplan.ics
```

**Outlook (web):** Kalender → Legg til kalender → Abonner fra web → lim
inn URL-en over.

**iPhone:** Innstillinger → Kalender → Kontoer → Legg til konto →
Annet → Legg til abonnert kalender → lim inn URL-en over.

### Merk om oppdateringsfrekvens

Kalenderabonnementer oppdateres ikke i sanntid. De fleste apper sjekker
for endringer hvert 12.–24. time. En endring i timeplanen kan derfor ta
opptil et døgn før den vises i kalenderen din.

### Innhold

Hver kalenderoppføring inneholder tidspunkt, klasse/fag og rom/sted.
Dette er offentlig undervisningsstruktur og inneholder ikke
personopplysninger om enkeltelever.
